# Delta Books Database System

A comprehensive SQL database solution developed for Delta Books, an educational book supplier, addressing cross-departmental business requirements from Finance, HR, and Database Administration departments.

## Features
- Relational database design for educational book supplier operations
- Structured SQL queries for multi-departmental business intelligence
- Comprehensive testing plans and quality assurance protocols
- Data validation and integrity checks
- Systematic query testing and optimization

## Technologies Used
- SQL Server
- T-SQL
- SSMS (SQL Server Management Studio)
- Database Design & Normalization
- Quality Assurance Testing
- Data Validation
